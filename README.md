# Basic Calculator (Java Swing)

A simple GUI-based calculator built using Java Swing that performs basic arithmetic operations.  
The application also handles invalid inputs and division by zero gracefully.

## 🚀 Features
- Performs Addition (+), Subtraction (-), Multiplication (X), Division (/)
- Built using Java Swing (GUI-based)
- Takes input through text fields
- Displays result in a separate text field
- Handles invalid input (letters or symbols)
- Prevents division by zero with proper message
- Includes "Clear" button to reset all fields

## 🛠️ Technologies Used
- Java
- Java Swing
- Event Handling (ActionListener with Lambda Expressions)

## ▶️ How to Run
1. Clone or download this repository
2. Open in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run `Calculator_Basic.java`

## ⚠️ Error Handling
- If user enters non-numeric input → displays "Invalid input!"
- If division by zero is attempted → displays "Cannot Divide by Zero"

## 📌 Project Structure
- `Calculator_Basic.java` → Main file containing GUI and logic
